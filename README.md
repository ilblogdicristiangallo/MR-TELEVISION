# MR-TELEVISION
IPTV media player for Fire TV OS 5–8. Play your own M3U playlists or Xtream servers. No content included.

# Description

MR-TELEVISION FREE is a media player for Fire TV OS (versions 5, 6, 7, and 8). Use it to play your own IPTV sources by adding M3U playlist URLs or connecting via Xtream Codes (server URL, username, and password).

Important: This application does not provide, host, or include any TV channels, movies, or streaming content. You must use sources you own or have permission to access.

Features:
- M3U playlist URL support
- Xtream Codes support
- TV-friendly interface for remote control
- External player option (if installed)
- Privacy: The app only connects to the servers/URLs you configure to download playlists and play streams. No personal data is collected for advertising purposes.

# MR-TELEVISION PRO
  Media player for Fire TV OS 5–8 with WireGuard engine. PRO version only via donation. VPN requires user config files.

# Description PRO
MR-TELEVISION PRO is a media player for Fire TV OS (versions 5, 6, 7, and 8) with an integrated WireGuard engine for VPN functionality. 

Important: This version is available **only via donation**. The app does not provide VPN servers, certificates, or configuration files. Users must supply their own WireGuard `.conf` files or `.zip` packages containing the configuration to activate VPN. 

The app also allows you to play your own M3U playlists or connect via Xtream Codes (server URL, username, password).

Features:

- Option to open streams with an external player

- Does not include VPN servers or certificates
- Only available via donation
